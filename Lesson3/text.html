Lesson 3

In this lesson, we're going to learn AJAX. AJAX means "Asynchronous JavaScript and XML," and it's really a web standard popularized by Google. It allows you to get data and send data on a website without refreshing the entire page all over again. You can just get and send the data you need as you need it.

There's a security issue if you just open the HTML file and it tries to run AJAX, so we need to spin up a basic local server. Go to where this repo is and run: 

python -m SimpleHTTPServer
Now in a browser, you can see the website at localhost:8000.

The default syntax is 
$.ajax({
  url: "http://...",
  method: "GET",
  success: function (data) {
    // do something with data
  }
});
